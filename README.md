## Getting Started

### yarn 安装

- NPM 安装
```bash
npm install -g yarn
```


- Mac OS 安装
```bash
curl -o- -L https://yarnpkg.com/install.sh | bash
```


- Windows 安装
```bash
https://yarnpkg.com/latest.msi
```

### yarn 使用

- Install dependencies
```bash
$ yarn
```

- Start the dev server
```bash
$ yarn start
```


- dist build
```bash
$ npm run build:prod
```

- zip 打包
```bash
$ npm run build:zip
```


- 错误
> EACCES: permission denied, unlink 'devScripts.ts'<br/>
> sudo chown -R 当前用户名 src
